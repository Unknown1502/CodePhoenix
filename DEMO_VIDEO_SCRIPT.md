## Technical Production Notes

### Tools Needed
- **Screen Recording**: OBS Studio or ScreenFlow
- **Video Editing**: DaVinci Resolve or Adobe Premiere
- **Motion Graphics**: After Effects (for phoenix animation)
- **Music**: Epidemic Sound or Artlist

### Recording Checklist
- [ ] Record at 1920x1080 60fps
- [ ] Use clean test data (COBOL sample file)
- [ ] Prepare mock analysis results (realistic data)
- [ ] Pre-generate transformation (for smooth demo)
- [ ] Test all animations work smoothly
- [ ] Record in one take per scene (easier editing)

### Editing Notes
- Fast-paced cuts (max 3 seconds per shot)
- Smooth transitions between scenes
- Text appears with motion (slide/fade)
- Music builds throughout
- Color grade: warm for modern code, cool for legacy
- Add subtle sound effects for interactions

### Narration
- **Voice**: Professional, energetic, confident
- **Pace**: Clear but energetic (not too fast)
- **Tone**: Problem ‚Üí Solution ‚Üí Impact
- **Recording**: Professional mic (Blue Yeti or better)

### Export Settings
- **Resolution**: 1920x1080
- **Frame Rate**: 60fps
- **Bitrate**: 20 Mbps
- **Format**: MP4 (H.264)
- **Audio**: AAC 320kbps
